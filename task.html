My first application using the React framework

*For a brief demo of this project check the video at the bottom of this page*

I wanted to learn React because I heard the process of creating a website becomes way more efficient once you pick up a framework. React seemed to be the most popular framework at the time so I went with it.

This app is a simple task tracker that utilizes React state/router, JSX, components/properties, and a locally run backend using JSON-server that enables the data to persist when the page is closed/reloaded.
